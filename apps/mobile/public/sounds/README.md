# Sound Effects Directory

Place sound effect files in this directory for use with overlay animations.

## Expected Files

-   `laser-zap.mp3` - Laser obliteration effect for clearing checked shopping list items

## File Format

Supported formats: MP3, WAV, OGG
Recommended: MP3 (best compatibility across platforms)

## Usage

Sound files are referenced in the animation effects library (`src/animations/effects.ts`) with paths like `/sounds/filename.mp3`.
